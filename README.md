SurelyWebRTC

A prebuilt Google WebRTC XCFramework for iOS, built from the official Google WebRTC source (144.7559.10) and distributed via CocoaPods.

Features

* Prebuilt Google WebRTC XCFramework
* Built from the official Google WebRTC source (144.7559.10)
* Easy integration with CocoaPods
* Supports iOS
* No functional modifications to the upstream WebRTC implementation

Requirements

* iOS 13.0+
* Xcode 16.0+
* Swift 5.9+

Installation

Add the following to your Podfile:

pod 'SurelyWebRTC'

Then run:

pod install
